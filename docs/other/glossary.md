# Базовые термины 

## Нода {id="node"}

Так называется устройство в Mesh сети. Цифровая рация. Каждая нода состоит из небольшой электронной платы с радиомодулем и антенны. Например **Heltec V3**, **Heltec V4**, **LilyGO T-Echo**, **heltec t114**

## Мобильная нода {id="mobil_node"}
Негабаритное карманное устройство которе не трудно **всегда** носить с собой.

Удобный корпус, стильный вид, небольшая антенна, [nRF52](/other/glossary#chips) на борту.

## Частота {id="freq"}

Физический канал, на котором устройства обмениваются данными. Представьте FM радиоприемник. Для перехода с _Радио Гача_ на _Европа плюс_ нужно покрутить ползунок и поймать волну. Так же и в Meshtastic, кроме той разницы, что у ноды нет такого ползунка и выбор диапазона чатот совершается при покупке ноды.    
Существуют безлицензионные диапазоны частот: 433 МГц, **868 МГц**, ~~915 МГц~~ и 2.4ГГц.    
Сеть города строится на **868 МГц**

## Антенна {id="ant"}

Антенна - часть отвечающая за преобразование электрических сигналов в радиоволны и обратно.

## HAM {id="ham"}

Лицензированный радиолюбитель.

# Железная часть

## Процессор {id="chips"}
Мозгом каждой ноды является процессор. Основные виды, применяемые в Mesh-сетях:
- nRF52 - Микропроцессор компании Nordic - один из лидеров энергоэффективных беспроводных решений.
Лучший вариант для мобильных нод. 3 дня работы от небольшого АКБ 18600 (1500mA) без режима энергосбережения.
Подключается по Bluetooth, USB    
- esp32 - Универсальная машина-убийца, мультитул в мире DIY. Куча разных прошивок, Bluetooth, WiFi.
Но и потребление в 10 раз больше. 
Лучший вариант для Сационарной ноды.   
Подключается по Bluetooth, Wifi, USB

## Технология LoRa

Специальный метод кодирования и передачи данных по радиоканалу, расшифровывается как Long Range (Большая Дальность).    
Особенность LoRa - способность принимать сигналы, которые по уровню мощности находятся ниже уровня шума радиоэфира.

## LoRa модуль {id="LoRamodules"}
